# API Configuration
Api configuration is handled in `/.env.local`.

## Keys

### Rate Limit
The number of requests allowed per minute.

Key: `RATE_LIMIT`

### API Path
The url of the remote llm api.

Key: `API_PATH`

### API Key
The authentication key for the API. Authorizes using the `Authorization : "Bearer "` header.

Key: `API_KEY`
